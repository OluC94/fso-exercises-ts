My server API

RESTful JSON API

Retrieve a hello message

GET /hello

Retrieve information about the number of phonebook entries

GET /info

Retrieve all people

GET /api/persons

Create a person

POST /api/persons

Attaching a JSON body with the possible below fields:

  • name (string)
  • number (string)

Retrieve a specific person

GET /api/persons/{id}

Delete a specific person

DELETE /api/persons/{id}
